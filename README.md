# aiteam-mkpasswd
Containerised Application For Creating Passwords and  SHA512 Hashes.

## Dependencies

- [Docker](https://docker.com "Homepage of docker")

## Getting Started

Pull the image:

``` bash
docker pull plone/aiteam-mkpasswd
```
